Graph Page

Permission required: Read


On the graph page, you can set the arguments "Time" and "EndDate". The first one, Time, is the time period

over which you want to graph data. If you want to graph the past week of data, then use "Time=1w". Valid abbreviations

for time are h, d, and w, which stand for hour, day, and week respectively. Only one can be used at a time. 1d2h

would return an error, but 26h is a valid constuction. The EndDate option manually specifies when to reference this time

period to. For example, by default, the graph will return the last week of data relative to the current day and time,

but specifying a prior date without changing the Time parameter will cause it to return the week of data prior to the

specified day. This query might look like "...EndDate=2023-08-04-23:59:00...". (Note: this is yyyy-mm-dd format.)

In summary, example queries might look like "/?ReadKey=...&Time=8d&EndDate=2023-08-03-12:00:00" which would return

a graph of the 8 days of data ending on August 3rd, 2023 at 12pm noon.

Image Page

Permission required: Read


This page returns an image from a camera in the greenhouse, if the camera exists.

Write Page

Permission required: Write


This page is the endpoint for write requests containing temperature data. Manually writing data is possible but rarely needed.

Therefore, this page has a separate access credential: the write key.
